Installed · First run

Turn open tabs into a working set.

Group matching sites now, then keep the result as a session you can reopen later.

01

Pin the extension

Open Chrome's puzzle menu and pin Group Tabs so the control stays in your toolbar.

02

Organize this window

Matching sites become groups. Pinned tabs, existing groups, and single-site tabs stay untouched.

03

Save the result

Open Group Tabs, name the window, and save it for later. Missing tabs restore into a new window.

Keyboard: Ctrl+Shift+G or ⌘Shift+G. Topic grouping is optional and runs through the browser's built-in on-device AI when available. Group Tabs has no cloud service.
